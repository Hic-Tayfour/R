Este é um repositório dedicado ao armazenamento de programas feitos em R no RStudio, com o objetivo de estudar as aplicações da biblioteca TidyVerse presente no R. Para isso, utilizei os vídeos de David Robinson, Tidy Tuesday R Screencasts, que são vídeos públicos disponíveis no YouTube no link: https://www.youtube.com/playlist?list=PL19ev-r1GBwkuyiwnxoHTRC8TTqP8OEi8. Aproveite bem o material aqui presente. Todas as bases de dados utilizadas aqui foram fornecidas por David Robinson nos vídeos.
